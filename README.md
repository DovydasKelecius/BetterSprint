# BetterMovement

BetterMovement is a lightweight client-side mod for Hytale that enhances player movement by improving sprinting behavior and overall movement responsiveness. The goal of this mod is to make movement feel smoother, more natural, and less restrictive compared to the vanilla experience without altering core gameplay balance.

This mod is ideal for players who want a more fluid movement system while keeping the game close to its intended design.

# Installation

To manually install the BetterMovement mod, download the `BetterMovement-vX.X.X.zip` file from the releases page and place it into your Hytale Mods folder. The default path is:

```
C:\Users\YourUser\AppData\Roaming\Hytale\UserData\Mods
```
